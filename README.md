## 🗞️ Descrição 

O Estaciona Fácil é um projeto desenvolvido para auxiliar donos de estacionamentos fictícios a gerenciarem seus negócios, oferecendo um sistema simples e intuitivo para controlar a entrada e saída de veículos e realizar o cálculo automático dos valores, tornando o processo mais rápido e eficiente.
